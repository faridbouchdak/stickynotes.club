# Product Model

> Status: v1

## Product promise

StickyNotes.club is a place where ideas can grow together.

## Lifecycle of an idea

Capture → Share → Discuss → Organize → Collaborate → Execute

## Public and Private

### Public

-   discover
-   inspire
-   share
-   attract collaborators

### Private

-   brainstorm
-   organize
-   collaborate
-   decide
-   execute

## Lifecycle of a note

New → Active → Enriched → Connected → Completed → Archived ↘ Deleted

## Lifecycle of a board

Created → Active → Shared → Completed → Archived ↘ Deleted

## Mental model

People should never need to understand the system before capturing an
idea.

## Product boundaries

StickyNotes.club is not: - project management software; - a document
editor; - a social network; - a knowledge base.

It sits between personal thinking and lightweight collaboration.
